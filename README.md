# 🕹️ PhotoPlay Masters IGO8: Technical Documentation

This repository serves as a technical knowledge base for the **PhotoPlay Masters IGO8** arcade system (specifically the **Version 2008 (ES) NSB IGO8 VM007** iteration).

The primary purpose of this project is to document and preserve the technical specifications of these legacy systems, including their system architecture and internal data structures.

> [!WARNING]

While efforts have been made to verify the data, there may be technical errors, hallucinations, or inconsistencies. Use this information as a reference only, and verify with hardware/binary data where possible.

---

## 📚 Technical Documentation

The repository currently focuses on the following specifications:

- **[System Architecture Guide](DOCS/SYSTEM_OVERVIEW.md)**: A deep dive into the PTS-DOS environment, the boot sequence, hardware detection (motherboards, sound chips), and touch interface logic.
- **[High Score (.HIS) Format](DOCS/HIS_FORMAT.md)**: A detailed binary specification of the high-score database used by Funworld games, including the custom 1990 epoch date system.
- **[Asset Container (.WAD) Format](DOCS/WAD_FORMAT.md)**: A technical breakdown of the encrypted resource archives, the XOR 0x55 cipher, and the internal file indexing system.

---

## 🎯 Objectives

1.  **Preservation**: Ensuring the technical logic of these machines is not lost to time.
2.  **Analysis**: Understanding how legacy POS hardware was adapted for arcade use in the late 90s and 2000s.
3.  **Community Reference**: Providing a starting point for others researching PhotoPlay hardware or similar PTS-DOS based systems.
